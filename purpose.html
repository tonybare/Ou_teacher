計畫目的

睿生光電股份有限公司(本計畫之合作廠商)為面板大廠群創(3481)新創之公司，投 資金額超過兩億元，為 X 光影像檢測科技為研發主體，由基板至整機，軟硬整合提供 一條龍客製化生產，具備整合 X 光供應鏈之能力，並且針對不同的 X 光應用，如：低 劑量醫療應用、動態 X 光等積極佈局。近期，睿生光電投入大量研發資源自主設計「三 層式雙能 X 光偵測技術」，提供高解析度骨頭、軟組織分離 X 光影像，其成像可檢測 過去遭受骨頭組織所遮蔽之病灶，可輔助醫生判斷病徵。然對於 X 光影像檢測之病徵 部分，如肺部的肺炎或肺結節，可利用深度學習技術協助醫生進行判讀，預期可有效 減輕醫生之工作負擔，因此，能夠以深度學習技術建構人工智慧醫學影像檢測方案成 為睿生光電股份有限公司之重點需求，即為本課程培育人才之重點。因應睿生光電的 研發需求方向，本學系所擬由大學部與碩士班現有的課程部分規劃配合之「人工智慧 影像檢測系統」課程模組以符合產業之需，創造 X 光影像檢測產業與人工智慧的發展 平台。此課程模組是在本系的原有課程規劃之下，由現有大學部課程中選出「人工智 慧導論」與碩士班課程中選出「高等人工智慧」等兩個課程作為課程模組的主軸，培 養學生對於人工智慧檢測系統設計之相關知識，以達成未來選修之學生至產業端實習 時可無縫接軌。
